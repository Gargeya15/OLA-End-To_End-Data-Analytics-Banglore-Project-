# 🚖 OLA Data Analyst Project  
### End-to-End Data Analytics Project | Python · SQL · Power BI  

---

## 📌 Overview  

This project is a complete end-to-end data analytics solution based on ride-booking data for Bengaluru city.  

It simulates real-world business scenarios of a ride-hailing platform like OLA and focuses on extracting actionable insights from raw data.  

The project covers the full analytics pipeline — from data cleaning and SQL analysis to building an interactive Power BI dashboard.  

---

## 🎯 Objectives  

- Analyze booking trends and ride patterns  
- Identify revenue distribution and key drivers  
- Understand cancellation behavior (customer & driver)  
- Evaluate performance across vehicle types  
- Build a business-ready dashboard for decision-making

## 📊 Dataset Information  

- **Total Rows:** ~100,000  
- **Location:** Bengaluru, India  
- **Time Period:** July 2024  
- **Vehicle Types:** Sedan, SUV, Mini, Auto, Bike, E-Bike, Prime Plus  
- **Booking Status:** Success, Cancelled, Incomplete  
- **Payment Methods:** Cash, UPI, Credit Card, Debit Card

## 🔄 Project Workflow  

### 1️⃣ Data Generation  
Synthetic dataset created using structured prompts  

### 2️⃣ Data Cleaning (Python)  
- Removed duplicates  
- Handled missing values  
- Standardized date/time formats  
- Validated categories  
- Converted data types  

### 3️⃣ SQL Analysis  
- Solved 10 business problems using MySQL  
- Created views for efficient querying  

### 4️⃣ Power BI Dashboard  
Built an interactive dashboard with 5 pages:  
- Overall  
- Vehicle Type  
- Revenue  
- Cancellation  
- Ratings

## 📈 Key Metrics  

- **Total Bookings:** 103,024  
- **Successful Bookings:** 63,967 (~62%)  
- **Total Revenue:** ₹35 Million  
- **High Cancellation Rate:** ~38%  
- **Average Ratings:** ~4.0  

---

## 📊 Dashboard Insights  

### 🚀 Overall  
- Stable ride demand throughout the month  
- Majority bookings are successful  

### 🚗 Vehicle Analysis  
- Prime Sedan → highest average distance  
- Auto → short-distance rides  
- E-Bike → strong booking value  

### 💰 Revenue Analysis  
- Cash is dominant payment method  
- UPI adoption is increasing rapidly  

### ❌ Cancellation Analysis  
- Top reason: Driver not moving towards pickup  
- Driver cancellations higher than customer cancellations  

### ⭐ Ratings Analysis  
- Ratings consistent across all vehicle types (~4.0)  
- Indicates stable service quality  

---

## 📊 Key Business Insights  

- High cancellation rate indicates operational inefficiency  
- Driver-related issues are the biggest problem  
- Different vehicle segments behave differently  
- Digital payments are growing  
- Strong consistency in service quality
